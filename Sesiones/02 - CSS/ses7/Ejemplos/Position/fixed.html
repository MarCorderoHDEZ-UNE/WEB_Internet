position: fixed

Esta propiedad es un tipo de posicionamiento absoluto, en donde la etiqueta padre del elemento se ignora y se toma la ventana del navegador como contenedor principal, posicionando los contenidos del elemento al cual se le asigne la propiedad en relación de las medidas del navegador.

Las propiedades left, right, bottom y top también pueden utilizarse para realizare el desplazamiento

En este caso, a la etiqueta h1 se le ha asignado un comportamiento de bloque (display: block) para darle medidas, se le ha puesto un 100% para el ancho de su contenido (width: 100%), se le dio una altura fija de 50px (height: 50px), se quitó el margen (margin: 0), se añadió relleno entre el contenido y el borde (padding: 20px), y para que permaneciera hasta arriba, se definió una posición en el tope de 0 (top: 0)
