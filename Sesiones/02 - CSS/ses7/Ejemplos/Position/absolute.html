position: absolute

Este valor permite posicionar el contenido de algún elemento de forma absoluta en relación a su contenedor padre (<body> si no hay ancestro), es decir, interrumpe el flujo normal de los elementos para otorgar una posición determinada fuera de todo contexto previo.

Si revisamos el código fuente de esta página, nos percataremos que la imagen que aparece del lado izquierdo está declarada entre este párrafo y el anterior, sin embargo, su posición es una completamente distinta a los esperado.

A la imagen se le está dando un ancho de 180px, se le declara una posición absoluta y luego se indica que estará pegada al tope de la página (top: 0) y con un desplazamiento a la derecha (lado derecho del navegador) de 25px

Intenta cambiar el tamaño de tu navegador para ver el comportamiento del elemento.
